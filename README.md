# HPGT_WSCOD 
# A Holistically Point-guided Text Framework for Weakly-Supervised Camouflaged Object Detection


# Clone repository
```
git clone git@github.com:moktsuiqin/HPGT_WSCOD.git
cd HPGT_WSCOD
```
